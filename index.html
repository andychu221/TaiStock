資料來源

此儀表板的所有資料皆來自您在此 GitHub 專案中 /data/ 資料夾內存放的 JSON 檔案。

網頁在載入時會自動讀取這些檔案，並將其用於所有圖表和表格的分析。這代表：

  • 資料是靜態的：儀表板顯示的數據內容取決於您上傳到 GitHub 的檔案。它不會自動抓取網路上的最新資料。
  • 更新方式：若要更新儀表板的數據，您需要手動下載最新的資料（例如，從證交所網站），將其轉換為對應的 JSON 格式，然後上傳並覆蓋 GitHub 儲存庫中 /data/ 資料夾內的舊檔案。
  • 資料格式：請確保 JSON 檔案的格式與原應用程式所產生或使用的格式一致，以避免解析錯誤。

如何維護您的資料庫？

您可以保留原版本的應用程式在本機端，專門用來擷取與下載最新的 HTML/JSON 檔案。當您取得新資料後，再透過 GitHub 的上傳功能，更新 /data/ 資料夾中的檔案即可。
